## ✨ المميزات

- 📶 **WiFi Direct** - نقل فائق السرعة بدون إنترنت
- 📲 **البلوتوث** - نقل آمن ومستقر
- 📡 **نقطة الاتصال** - إنشاء شبكة خاصة للمشاركة
- 🚀 **سرعة عالية** - حتى 40 ميجابايت/ثانية
- 🔒 **آمن** - تشفير أثناء النقل
- 📁 **جميع أنواع الملفات** - صور، فيديو، تطبيقات، مستندات
- 🌙 **واجهة عصرية** - تصميم Material 3
